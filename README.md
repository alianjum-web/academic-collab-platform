# Academic Networking Platform

An open-source academic collaboration platform for students, alumni, and faculty.

## Features
- Role-based dashboards
- Secure authentication
- Real-time academic discussions
- Industry-verified insights

## Tech Stack
- Node.js
- React / Next.js
- Firebase
- PostgreSQL

## Installation
```bash
git clone https://github.com/USERNAME/academic-networking-platform.git
cd academic-networking-platform
npm install
npm run dev
